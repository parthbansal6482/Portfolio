# Product

## Register

brand

## Users
Recruiters, engineering managers, and technical peers. They are evaluating technical capabilities, design craft, and visual taste under brief attention spans.

## Product Purpose
A personal portfolio showcasing high-end visual web development, robust system engineering, and elegant interactive animations to stand out as a highly creative, skilled frontend and systems engineer.

## Brand Personality
Elegant, technical, and organic. A creative, high-performing developer portfolio that blends code precision with interactive, natural warmth (like the flower seed sprout cursor).

## Anti-references
Cliché SaaS templates, boring rectangular grids, generic navy/gold palettes, and standard uninspired layouts.

## Design Principles
- **Editorial Sophistication**: Elegant typography hierarchy and generous breathing room.
- **Organic Interactions**: Flowing animations, smooth parallax scroll shifts, and natural cursor micro-interactions (seed sprouting).
- **Engineering Craft**: Production-ready code structure, responsive adaptability, and flawless runtime performance.

## Accessibility & Inclusion
- Native WCAG 2.1 AA color contrast compliance.
- Strict reduced-motion media query fallback support for all custom motion paths.
